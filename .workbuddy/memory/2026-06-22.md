# 2026-06-22 工作记录

## 项目骨架搭建

基于 `MediaMusic.md`（PRD）搭建了 MediaMusic 本地无损音乐播放器的完整项目骨架。

### 技术栈
- **前端 UI**：Photino.Blazor 4.0.13（Blazor Razor 组件）
- **目标框架**：.NET 10.0（SDK 10.0.201，PRD 中"后续 LTS 版本"）
- **音频引擎**：ManagedBass 4.0.2 + .Mix/.Fx/.Flac/.Aac（原生 DLL 需手动从 un4seen 下载放入 native/bass/）
- **数据库**：Microsoft.Data.Sqlite 8.0.28 + Dapper 2.1.66
- **CSS**：Tailwind v3.4.17 + PostCSS + Autoprefixer（Frontend/ 工具链）

### 项目布局
单项目 + 文件夹组织，无 src/ 嵌套。关键目录：
- `Audio/` — BASS 引擎初始化、播放、EQ、效果
- `Data/` — SQLite schema + POCO + Dapper 仓储
- `Library/` — 双模曲库（拖拽即播 + 深度扫描）、元数据读/写
- `Lyrics/` — LRC 解析
- `Platform/` — Win32 P/Invoke（全局热键、托盘、窗口拖拽、点击穿透）
- `Services/` — DI 注册、主题切换、应用状态、设置
- `Layout/` — 无边框主布局 + 自定义标题栏
- `Pages/` — 可路由视图（主界面、曲库、均衡器、元数据编辑）
- `Windows/` — 迷你播放器、桌面歌词
- `Components/` — 播放控件、EQ 面板、虚拟化曲目列表、封面组件

### 关键决策（用户确认）
- 用户选择 Photino.Blazor 而非纯 HTML
- 单项目 + 文件夹组织
- 需要 Tailwind 前端构建工具链
